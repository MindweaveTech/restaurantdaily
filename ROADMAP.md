# Restaurant Daily - Product Roadmap

## Vision
A comprehensive restaurant operations management platform that simplifies daily cash tracking, team management, and financial reporting for Indian restaurants.

---

## Phase 1: Foundation (Current) ✅
*Core authentication and basic structure*

- [x] Phone-based OTP authentication (Twilio SMS)
- [x] Role-based access control (Admin/Staff)
- [x] Restaurant onboarding flow
- [x] Dark glassmorphism UI theme
- [x] Mobile-first responsive design
- [x] Supabase database integration
- [x] Demo login for testing

---

## Phase 2: Core Operations (In Progress)
*Daily operations tracking and management*

### Attendance System
- [ ] Staff check-in/check-out
- [ ] Real-time attendance dashboard

### Cash Session Management
- [ ] Opening/closing balance tracking
- [ ] Cash handover workflow
- [ ] Session notes and discrepancy flagging
- [ ] Daily reconciliation reports

- [ ] Voucher creation with categories
